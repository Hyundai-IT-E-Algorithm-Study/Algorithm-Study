<그래프 탐색: 어떤 것들이 연속해서 이어질 때, 모두 확인하는 방법>
2중 for문을 돌려 값이 1이고 방문 안했을 때만 계속 탐색
=> boolean을 사용하여 false로 초기화하고 방문하면 true로 바꿔준다.
BFS
1. 너비 우선 탐색
2. 자기 바로 밑 자식들을 우선 탐색
3. 우선 시작점에 연결된 자식들(vertex)를 찾는다.
4. 찾은 vertex를 queue에 저장한다.
5. 시간복잡도: O(vertex+edge) => 여기서 edge는 vertex를 연결하고 있는 선과 같은 개념
6. 코딩 구조: 검색할 그래프, 그래프 방문여부 확인, bfs를 실행할 queue 필요
7. 최단 거리 찾기(미로 찾기)문제

DFS
1. 깊이 우선 탐색
2. 자기의 자식 -> 자기의 자식의 자식 -> 더 이상 자식이 없으면 다시 올라와서 그것의 자식 탐색
3. 스택, 재귀함수 주로 사용 v -> 재귀함수 쓰는 게 코드 길이가 더 짧아짐
4. 경로의 특징을 저장해두는 문제
